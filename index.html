Masukkan Gemini API Key Anda

Untuk menggunakan fitur AI, Anda memerlukan Gemini API Key. Dapatkan kunci Anda secara gratis dari Google AI Studio.

  1. Buka Google AI Studio.
  2. Login dengan akun Google Anda.
  3. Klik tombol "Get API key".
  4. Buat kunci API baru di proyek Anda.
  5. Salin kunci API yang baru dibuat dan tempel di bawah ini.
